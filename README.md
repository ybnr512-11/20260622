# 로또 추첨기 (Lotto Picker)

한국 로또 6/45 규칙에 맞는 웹 기반 무작위 번호 생성기입니다.

## 실행 방법

`index.html` 파일을 브라우저에서 열면 됩니다. 별도 설치가 필요 없습니다.

## 기능

- 1~45 번호 중 6개 + 보너스 번호 무작위 추첨
- TV 방송 스타일 추첨기 애니메이션 (혼합 구 · 출구 · 당첨번호판)
- 즉시 생성, 1/3/5게임, 추첨 기록
- HP 디자인 시스템 (`DESIGN.md`) 반영 UI
